Project Site Plan

Site Name

Name: Bay Area eBiking Club

Reason: This name represents a club focused on electric biking in the San Francisco Bay Area.

Site Purpose

The purpose of this site is to provide a hub for regional e-biking by offering trail recommendations, e-bike reviews, and membership forms.

Scenarios

  • What is the best e-bike to get for mountainous areas?
  • Where can I find contact information for the club's directors?

Color Schema

  • Color 1: Blue (#3498db) - Used for headers and links.
  • Color 2: Light Gray (#ecf0f1) - Used for the background.

Typography

  • Font 1: Arial - Used for headings.
  • Font 2: Verdana - Used for body text.
